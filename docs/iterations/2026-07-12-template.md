# Initial Template

Created the Java 21 Spring Boot skeleton with Maven, PostgreSQL, Flyway, Kafka, Docker Compose, ports-and-adapters guidance, DDD/TDD conventions, agent instructions, skills, and a Product CRUD spec.

Implemented the first Animal Action slice: API-first animal registration and action execution, PostgreSQL persistence, Kafka event publication/listening, a feature spec, and domain tests. Direct Kafka publication remains the documented local delivery baseline.

Established the feature workflow as API/event-contract first with TDD: specification and contract, failing tests, implementation, adapters, verification, and documentation.
